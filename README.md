# АвтоМаг

# Описание проекта
Данный проект представляет собой сервис по продаже машин.
На сайте располагаются объявления. 

Объявлении состоит:
- описание
- марка машины
- тип кузова
- фото
- статус о продаже

# Стек технологий
- Java 17
- SpringBoot 2.7.4
- Checkstyle-plugin 3.1.2
- Liquibase 4.15.0
- Hibernate 5.6.11
- PostgreSQL 14
- Lombok 1.18.22
